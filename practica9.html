Unidades de Medida en CSS

Esta página muestra ejemplos de diferentes unidades de medida absolutas y relativas utilizadas en CSS. Puedes experimentar cambiando los valores en el código CSS (en la sección <style>) para ver cómo afectan a la presentación de los elementos.

Unidades de Medida Absolutas

Las unidades absolutas tienen un tamaño fijo y no dependen de ningún otro elemento o configuración. Generalmente se recomiendan para medios impresos o cuando se necesita una precisión absoluta.

px (píxeles)

Representan un punto en la pantalla (aunque la densidad de píxeles puede variar entre dispositivos).

100px x 50px

Prueba a cambiar: El width a 150px o el background-color a lightblue.


.example-box-absolute {
    width: 100px; /* Intenta cambiar a 150px */
    height: 50px;
    background-color: lightcoral; /* Intenta cambiar a lightblue */
    color: white;
    text-align: center;
    line-height: 50px;
    font-size: 16px;
    margin-bottom: 10px;
}

Unidades de Medida Relativas

Las unidades relativas basan su tamaño en otro valor, como el tamaño de la fuente del elemento padre o del elemento raíz (<html>), o el tamaño de la ventana del navegador.

em

Relativo al tamaño de la fuente del elemento padre.

Texto con tamaño base
Texto 1.5em (24px si el padre es 16px)

Prueba a cambiar: El font-size del elemento padre (.example-box-relative-em) a 20px y observa cómo afecta al hijo.


.example-box-relative-em {
    font-size: 16px; /* Intenta cambiar a 20px */
}

.example-box-relative-em .child-em {
    font-size: 1.5em;
    background-color: lightseagreen;
    color: white;
    padding: 5px;
}

rem

Relativo al tamaño de la fuente del elemento raíz (<html>).

Texto con tamaño base (depende del <html>)
Texto 1.2rem (si el html tiene 16px, esto será 19.2px)

Prueba a cambiar: El font-size del selector html en la parte superior del <style> a 20px y observa el cambio.


html {
    font-size: 16px; /* Intenta cambiar a 20px */
}

.example-box-relative-rem .child-rem {
    font-size: 1.2rem;
    background-color: lightskyblue;
    color: white;
    padding: 5px;
}

vw y vh

Relativo al 1% del ancho (vw) y la altura (vh) de la ventana del navegador.

50vw x 20vh

Prueba a cambiar: Los valores de width y height a 70vw y 30vh respectivamente, y redimensiona la ventana del navegador.


.example-box-relative-vwvh {
    width: 50vw; /* Intenta cambiar a 70vw */
    height: 20vh; /* Intenta cambiar a 30vh */
    background-color: lightgoldenrodyellow;
    text-align: center;
    line-height: 20vh;
}

% (porcentaje)

Relativo al tamaño del elemento padre.

50% x 50%

Prueba a cambiar: El width del padre (.example-box-relative-percent-parent) a 300px o el background-color del hijo a purple.


.example-box-relative-percent-parent {
    width: 200px; /* Intenta cambiar a 300px */
    height: 100px;
    background-color: #ccc;
    position: relative;
    margin-bottom: 10px;
}

.example-box-relative-percent-child {
    width: 50%;
    height: 50%;
    background-color: orange; /* Intenta cambiar a purple */
    position: absolute;
    top: 25%;
    left: 25%;
    text-align: center;
    line-height: 50px;
    color: white;
}
